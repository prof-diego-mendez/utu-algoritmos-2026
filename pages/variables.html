01 — Concepto

¿Qué es una variable?

Una variable es un espacio con nombre en la memoria del programa donde se puede guardar un dato. Pensá en ella como una caja etiquetada: la etiqueta es el nombre y lo que hay adentro es el valor.

🏷️
Nombre
Identificador único que usamos para referir a la variable en el código. Debe ser descriptivo.
📦
Valor
El dato almacenado dentro de la variable. Puede cambiar durante la ejecución.
🔖
Tipo
La clase de dato que contiene: número, texto, valor lógico, etc.

Declaración y asignación

Declarar es crear la variable. Asignar es guardar un valor en ella.

declaracion.pseudo 
// Declarar y asignar en el mismo paso
edad     ← 25
nombre   ← "Ana"
promedio ← 8.5
aprobado ← verdadero

// Modificar el valor de una variable existente
edad ← 26           // ahora edad vale 26

// Mostrar en pantalla
escribir(nombre)    // → Ana
escribir(edad)      // → 26

Entrada del usuario

entrada.pseudo 
escribir("¿Cuál es tu nombre?")
nombre ← leer()
escribir("Hola, " + nombre + "!")



02 — Tipos de datos

¿Qué puede guardar una variable?

El tipo de dato determina qué clase de información puede almacenar la variable y qué operaciones se pueden hacer con ella. Básicamente se pueden almacenar 4 tipos de datos: enteros, reales, texto y booleanos.

Tipo Ejemplos de valores Descripción Uso típico
entero 42   -7   0   1000 Números sin parte decimal Edades, contadores, índices
real 3.14   -0.5   9.99 Números con parte decimal Precios, promedios, medidas
texto "Hola"   "abc"   "42" Cadena de caracteres Nombres, mensajes, códigos
lógico verdadero   falso Solo dos posibles valores Condiciones, estados, flags
tipos.pseudo 
cantidad  ← 10              // entero
precio    ← 19.99           // real
producto  ← "Cuaderno"      // texto
enStock   ← verdadero       // lógico

total ← cantidad × precio  // → 199.90
escribir(producto + ": $" + total)

operaciones.pseudo 
a ← 10
b ← 3

suma      ← a + b      // → 13
resta     ← a - b      // → 7
producto  ← a × b      // → 30
cociente  ← a ÷ b      // → 3.333...
resto     ← a mod b    // → 1
potencia  ← a ^ b      // → 1000

// Acumulador: sumar al valor anterior
total ← 0
total ← total + 5    // → 5
total ← total + 8    // → 13
total ← total + 12   // → 25

Un ejemplo clásico de operaciones entre variables: Intercambio de variables

Para intercambiar los valores de dos variables necesitamos una tercera variable auxiliar.

intercambio.pseudo 
x   ← 5
y   ← 10
aux ← x     // guardamos x en aux → aux = 5
x ← y              // x toma el valor de y → x = 10
y ← aux            // y toma el valor original de x → y = 5

escribir("x = " + x)  // → x = 10
escribir("y = " + y)  // → y = 5
